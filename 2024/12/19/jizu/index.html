时钟同步方法

tsetup

建立时间是指在时钟边沿到来之前，数据输入（D）信号需要保持稳定的最小时间间隔。
如果数据未能在这个时间内稳定，触发器可能无法正确捕获数据，导致计数器或同步逻辑工作异常。

tccq

Clock-to-Q Delay
时钟到输出延迟是指从触发器接收到时钟边沿信号开始，到触发器输出 Q 信号发生变化所需的时间。
它反映了触发器的内部传播延迟。

1. R 型指令

R 型指令的操作码固定为 000000，具体的操作由功能码（funct）决定。

指令 功能 opcode rs rt rd shamt funct
add 加法 000000 任意 任意 任意 00000 100000
addu 无符号加法 000000 任意 任意 任意 00000 100001
sub 减法 000000 任意 任意 任意 00000 100010
subu 无符号减法 000000 任意 任意 任意 00000 100011
and 按位与 000000 任意 任意 任意 00000 100100
or 按位或 000000 任意 任意 任意 00000 100101
xor 按位异或 000000 任意 任意 任意 00000 100110
nor 按位或非 000000 任意 任意 任意 00000 100111
slt 小于则置位 000000 任意 任意 任意 00000 101010
sltu 无符号小于则置位 000000 任意 任意 任意 00000 101011
sll 逻辑左移 000000 00000 任意 任意 任意 000000
srl 逻辑右移 000000 00000 任意 任意 任意 000010
sra 算术右移 000000 00000 任意 任意 任意 000011
jr 跳转到寄存器地址 000000 任意 00000 00000 00000 001000

2. I 型指令

I 型指令通过操作码（opcode）区分功能，不需要功能码。

指令 功能 opcode rs rt immediate / offset
addi 加立即数 001000 任意 任意 16位立即数
addiu 无符号加立即数 001001 任意 任意 16位立即数
andi 按位与立即数 001100 任意 任意 16位立即数
ori 按位或立即数 001101 任意 任意 16位立即数
xori 按位异或立即数 001110 任意 任意 16位立即数
slti 小于立即数置位 001010 任意 任意 16位立即数
sltiu 无符号小于立即数置位 001011 任意 任意 16位立即数
lw 加载字 100011 任意 任意 基地址 + 偏移量
sw 存储字 101011 任意 任意 基地址 + 偏移量
lui 加载高位立即数 001111 00000 任意 16位立即数
beq 相等分支 000100 任意 任意 偏移量
bne 不等分支 000101 任意 任意 偏移量

3. J 型指令

J 型指令通过操作码（opcode）区分功能，不需要功能码。

指令 功能 opcode 地址 (26位)
j 跳转 000010 目标地址
jal 跳转并链接 000011 目标地址
