函数 功能
torch.abs(A) 绝对值
torch.add(A, B) 相加，A 和 B 既可以是 Tensor 也可以是标量
torch.clamp(A, max, min) 裁剪，A 中的数据若小于 min 或大于 max，则变成 min 或 max，即保证范围在 [min, max]
torch.div(A, B) 相除，A % B，A 和 B 既可以是 Tensor 也可以是标量
torch.mul(A, B) 点乘，A * B，A 和 B 既可以是 Tensor 也可以是标量
torch.pow(A, n) 求幂，A 的 n 次方
torch.mm(A, B.T) 矩阵叉乘，注意与 torch.mul 之间的区别
torch.mv(A, B) 矩阵与向量相乘，A 是矩阵，B 是向量，这里的 B 是否需要转置都可以
A.item() 将 Tensor 转化为基本数据类型，注意 Tensor 中只有一个元素时才可以使用，常用于在 Tensor 中取出数值
A.numpy() 将 Tensor 转化为 Numpy 类型
A.size() 查看尺寸
A.shape 查看尺寸（等同于 A.size()）
A.dtype 查看数据类型
A.view() 重构张量尺寸，类似于 Numpy 中的 reshape
A.transpose(0, 1) 行列交换
A[1:] 切片操作
A[-1, -1] = 100 赋值操作，类似于 Numpy 中的切片
A.zero_() 归零化
torch.stack((A, B), dim=-1) 拼接，升维
torch.diag(A) 取 A 对角线元素形成一个一维向量
torch.diag_embed(A) 将一维向量放到对角线中，其余数值为 0 的 Tensor

nn.Module

神经网络模型是通过继承 nn.Module 类来定义的。forward 方法用于定义网络的前向传播。

优化器(Optimizer)

PyTorch 提供了多种优化器（如 SGD, Adam 等）来优化神经网络的参数.
优化器的作用是更新神经网络的权重，以最小化损失函数。优化器通过计算梯度来更新权重，从而逐步逼近最优解。
我们一般就用Adam和SGD比较多

SGD

  • 适合小型或简单任务
  • 可以添加动量（momentum），改善优化路径的平滑性，有助于更快地收敛。

训练循环（Training Loop）

Adam

  • 自适应调整学习率，收敛速度快。
  • 对初始学习率不敏感，适合复杂模型和大规模数据集。

数据加载(DataLoader)

PyTorch 的 DataLoader 用于批量加载数据，通常与 Dataset 类结合使用来管理和加载数据集。

torch.nn

PyTorch 中的核心模块，提供了大量用于构建神经网络的功能,以下是常用的：

  • torch.nn.linear：全连接层

  • torch.nn.Conv1d, Conv2d, Conv3d: 一维、二维、三维卷积。

  • 池化

    torch.nn.MaxPool1d, MaxPool2d, MaxPool3d: 最大池化。
    torch.nn.AvgPool1d, AvgPool2d, AvgPool3d: 平均池化。
    torch.nn.AdaptiveAvgPool2d: 自适应平均池化，可将输出调整为指定大小。

  • 归一化

    torch.nn.BatchNorm1d, BatchNorm2d: 批量归一化，减轻协变量偏移。
    torch.nn.LayerNorm: 层归一化。
    torch.nn.InstanceNorm2d: 实例归一化，常用于风格迁移。

  • MultiheadAttention

    embed_dim: 输入序列中每个词的嵌入维度（即特征维度）。
    num_heads: 注意力头的数量。embed_dim 必须可以被 num_heads 整除。
    dropout (默认：0.0): 在注意力权重上的 dropout 比例。
    bias (默认：True): 是否在注意力计算中使用偏置。
    batch_first (默认：False): 如果为 True，输入的第一个维度是 batch_size，否则是序列长度。

  • torch.nn.Embedding嵌入层

    num_embeddings: 嵌入矩阵的词表大小，即总共可以嵌入多少个索引。
    embedding_dim: 每个嵌入向量的维度。
    padding_idx (默认：None): 如果指定，该索引的嵌入向量会初始化为零，且不会被梯度更新。
    max_norm (默认：None): 如果指定，嵌入向量的 L2 范数会被约束到该值。
    scale_grad_by_freq (默认：False): 是否根据索引频率对梯度进行缩放。

  • torch.nn.Parameter

    一种特殊的张量，用于表示模型的可学习参数（即在训练中需要优化的权重）
    torch.nn.Parameter 是 torch.Tensor 的子类。
    与普通张量不同，当 Parameter 被赋给 torch.nn.Module 的属性时，会自动添加到模型的参数列表中，参与梯度计算和优化。

  • 激活函数

    • torch.nn.ReLU
    • torch.nn.Sigmoid
    • torch.nn.Tanh
    • 等等

  • 损失函数

    • 回归损失

      torch.nn.MSELoss: 均方误差（Mean Squared Error）。
      torch.nn.L1Loss: 平均绝对误差（Mean Absolute Error）。
      torch.nn.SmoothL1Loss: 平滑的 L1 损失。

    • 分类损失

      torch.nn.CrossEntropyLoss: 交叉熵损失，适用于多分类任务。
      torch.nn.BCELoss, BCEWithLogitsLoss: 二元交叉熵损失。
      torch.nn.NLLLoss: 负对数似然损失，通常和 LogSoftmax 一起用。
